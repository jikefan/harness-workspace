# Projects

Project task workspaces live here.

Use:

```text
projects/<project>/active/<task>/
projects/<project>/archived/<YYYY-MM>/<task>/
projects/<project>/scratch/<experiment>/
```

Public example registry:

- `projects/example-product/tasks.yml`: anonymized sample task registry.

Real task registries and working copies are ignored by default because they can reveal roadmap, branch names, PR numbers, people, customer context, and private code paths.
